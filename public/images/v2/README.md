# v2 画像アセット

このフォルダには TOP ページ v2 用の画像を配置します。

## 配置済み（既存画像からコピー）

- `hero_top.png` … ヒーロー画像
- `ranking-1.png`, `ranking-2.png`, `ranking-3.png` … ランキングアイコン
- `user-icon-1.png`, `user-icon-2.png` … ユーザーアイコン（user-1.png, user-2.png からコピー）

## 手動で追加する画像

アップロードされた以下の2つは、手動でこのフォルダに配置してください。

- `recommend-banner.png` … おすすめバナー
- `school-logo.png` … スクールロゴ（Tech>Go 等）

これらは Cursor のアップロード先から `public/images/v2/` にコピーするか、デザインから書き出して配置してください。
