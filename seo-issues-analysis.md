# SEO観点での改善ポイント

全ページの meta title・description を分析し、SEOに不利になり得る箇所をまとめました。

---

## 1. 重大：タイトルの重複・類似（未経験者マガジン）

### 問題
7ページが**ほぼ同じタイトル構造**を使用しています。

| パス | タイトル（前半部分） |
|------|---------------------|
| 20s-women | 20代女性の未経験転職｜**成功のポイントとおすすめ業界・職種** |
| 30s | 30代の未経験転職｜**成功のポイントとおすすめ業界・職種** |
| 30s-women | 30代女性の未経験転職｜**成功のポイントとおすすめ業界・職種** |
| 40s | 40代の未経験転職｜**成功のポイントとおすすめ業界・職種** |
| 40s-women | 40代女性の未経験転職｜**成功のポイントとおすすめ業界・職種** |
| high-school | 高卒・高校中退の就職｜**成功のポイントとおすすめ業界・職種** |
| high-school-women | 高卒女性の就職｜**成功のポイントとおすすめ業界・職種** |

### リスク
- 検索結果でページの違いが分かりにくい
- タイトルの独自性が低く、差別化しづらい
- 同一サイト内でのキーワード競合の可能性

### 推奨
各ページの**内容に合わせた固有のフレーズ**に変更する。

例：
- 20s-women: 「20代女性の未経験転職｜スキルなしでも叶うキャリアチェンジ術」
- 30s: 「30代未経験転職｜ポータブルスキルを活かす転職成功法」
- 40s: 「40代未経験転職｜社会人経験を武器にする転職術」
- high-school: 「高卒・高校中退の就職｜未経験から挑戦できる業界と就活のコツ」
- high-school-women: 「高卒女性の就職｜ワークライフバランスを重視した就職ガイド」

---

## 2. 中程度：タイトルの長さ

### 問題
多くのページで「\| 手取り計算シミュレーションツール テドリさん」が付き、**タイトルが長くなりすぎている**可能性があります。

- 日本語の目安：約30〜35文字で切られることが多い
- 例：`転職したい20代未経験者必見！ゼロから始める理想のキャリアパス | 手取り計算シミュレーションツール テドリさん` → 50文字超

### 推奨
- ブランド名を短くする：「\| テドリさん」のみにする
- または、layout の template `%s｜テドリさん` を活用し、ページ側の title からブランド名を省く

---

## 3. 中程度：Description の類似

### 問題
未経験者マガジンの Description が似通っています。

**女性向け（20s/30s/40s-women）の共通パターン：**
- 「XX女性の未経験転職の現状、ワークライフバランスを重視した転職のコツを解説」

**年代別（30s/40s）の共通パターン：**
- 「XX代の未経験転職の現状、…即戦力として評価される転職のコツを解説」

### リスク
- 検索結果の説明文が似ており、クリック率が下がりやすい
- ページごとの独自性が伝わりにくい

### 推奨
各ページの**強み・特徴**を具体的に盛り込む。

例：
- 20s-women: 「20代女性の未経験転職。スキルなしでもOKの業界、履歴書の書き方、面接対策まで。ワークライフバランスを重視した転職のコツを解説。」
- 40s-women: 「40代女性の未経験転職。社会人経験の活かし方、ポータブルスキルの伝え方、ミドル向け求人の探し方を解説。」

## 4. 軽微：キャリアページのブランド名

### 問題
`/career/career-change-20s` と `/career/high-income` のタイトルに「テドリさん」が含まれていません。

- 20代転職の年収相場｜平均年収365万円から年収アップする方法
- 年収1000万超えの手取り早見表｜税金・社会保険料の内訳を解説

### リスク
- 他サイトとの区別がしづらい
- ブランド認知の機会を逃す

### 推奨
末尾に「｜テドリさん」を追加する。

---

## 5. 軽微：年度表記の陳腐化

### 問題
`/tools/job-change` のタイトルに「【2026年版】」が入っています。

### リスク
- 2027年以降、古い印象を与える
- 毎年更新が必要になる

### 推奨
- 「【2026年版】」→「【最新版】」に変更する
- または、年度ごとにメタ情報を更新する運用にする

---

## 6. 軽微：早見表トップの Description

### 問題
`/tables` の Description が短めです。

「年収200万円〜1000万円、年代別の手取り額を一覧表で確認できます。」（約40文字）

### 推奨
もう少し情報を足して、クリックを促す。

例：「年収200万円〜1000万円の手取り額を年代別に一覧表示。所得税・住民税・社会保険料の内訳も確認できる早見表。完全無料。」

---

## 7. 軽微：キーワード競合の可能性

### 問題
「20代 転職」「未経験 転職」などが複数ページで重なっています。

- `/career/career-change-20s`：20代転職の年収相場
- `/magazine/inexperienced/20s`：20代未経験者の転職

### リスク
- 同一サイト内で同じキーワードを狙い、評価が分散する可能性

### 推奨
- career-change-20s：**年収・手取り**寄り（「20代 年収」「20代 手取り」）
- inexperienced/20s：**転職ノウハウ**寄り（「20代 未経験 転職」「第二新卒」）
- 狙うキーワードを分けて、タイトル・Description を調整する

---

## 優先度まとめ

| 優先度 | 項目 | 影響 |
|--------|------|------|
| 高 | 未経験者マガジンのタイトル重複 | 検索結果での差別化、クリック率 |
| 中 | タイトル長さ・ブランド名 | 表示切れ、ブランド認知 |
| 中 | Description の類似 | クリック率 |
| 低 | キャリアページのブランド名 | ブランド認知 |
| 低 | 2026年版表記 | 陳腐化リスク |
| 低 | 早見表 Description | クリック率 |
| 低 | キーワード競合 | 評価分散の可能性 |
