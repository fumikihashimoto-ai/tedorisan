# 「万円」ラベル表示差の調査レポート

## 現象
「あなたの月収」入力欄内の右側に表示される「万円」が、「年代を選択」（select 未選択時）と異なる表示になっている。

---

## 要因

### 実装の比較

| 項目 | 年代を選択（select 未選択時） | 万円（span） |
|------|-------------------------------|--------------|
| 要素 | `<select>` の表示テキスト | `<span>`（absolute 配置） |
| フォント | `font-['Noto_Sans_JP']` | 未指定（body 継承） |
| 字重 | `font-bold` (700) | 未指定（normal = 400） |
| サイズ | `text-[14px]` | `text-[12px]` |
| 色 | `text-[#AAAAAA]` | `text-gray-600`（#4b5563） |

### 結論
「万円」は**プレースホルダー統一仕様（Noto Sans JP・Bold・14px・#AAAAAA）が適用されていない**ため、年代 select と見た目がずれている。

- **色**: gray-600（#4b5563）→ やや濃いグレー。統一仕様は #AAAAAA。
- **サイズ**: 12px → 統一仕様は 14px。
- **字重**: 指定なし（normal）→ 統一仕様は Bold。

---

## 改善案

「万円」をプレースホルダー統一仕様に合わせる。

- **フォント**: Noto Sans JP（`font-['Noto_Sans_JP']` または継承のまま）
- **字重**: Bold（`font-bold`）
- **サイズ**: 14px（`text-[14px]`）
- **色**: #AAAAAA（`text-[#AAAAAA]`）
- **配置**: 右寄せのまま（`text-right`）

これで「年代を選択」「あなたの月収」の placeholder と「万円」の三つが同一仕様で表示される。
